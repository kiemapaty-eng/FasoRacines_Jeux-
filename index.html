Confidentialité

Dernière mise à jour : juillet 2026. Racines est une petite application indépendante ; ce texte décrit simplement et honnêtement ce que nous faisons de vos données.

Données collectées

  • Si vous créez un compte : votre email, votre mot de passe (jamais stocké en clair, géré par Firebase Authentication), et le nom que vous choisissez d'afficher.
  • Vos statistiques de jeu (parties jouées, victoires) liées à votre compte.
  • En mode "Jouer en ligne" sans compte, seul un code de salle temporaire est créé ; il est supprimé automatiquement et n'est lié à aucune identité.

Utilisation

  • Ces données servent uniquement à faire fonctionner votre profil, vos statistiques et le multijoueur.
  • Nous ne vendons ni ne partageons vos données avec des tiers à des fins publicitaires. Il n'y a pas de publicité dans Racines.

Stockage

Les données sont hébergées par Firebase (Google) avec des règles d'accès restreignant chaque profil à son propriétaire. Vous pouvez demander la suppression de votre compte et de vos données à tout moment.

Enfants

Racines peut être utilisée en famille. Nous recommandons aux parents de superviser la création de compte pour les plus jeunes joueurs.

Contact

Pour toute question ou demande de suppression de données, contactez la développeuse via la page GitHub du projet.
